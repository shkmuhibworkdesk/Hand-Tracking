 AI Sign Language Detection System

A real-time sign language recognition system using computer vision that converts hand gestures into spoken words.

 Features

* Real-time hand detection
* Gesture classification
* Transcript generation
* Speech output using text-to-speech

 Technologies

Python
OpenCV
MediaPipe
NumPy
pyttsx3

 How it Works

1. Webcam captures video input
2. MediaPipe detects hand landmarks
3. Gesture classifier determines hand sign
4. Detected gestures are added to a transcript
5. Transcript is read aloud using TTS
