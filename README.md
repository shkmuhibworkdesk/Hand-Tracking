This repository contains all the files necessary to create a hand-tracking project. 
This also includes an implementation plan outlining the step-by-step process.
If you wish to use this project, make sure to downgrade your Python version to 3.9 or 3.10.0.
Install libraries of mediapipe and opencv 2 using commands "pip install mediapipe opencv-python".
